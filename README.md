# نظام إدارة الدوام والرواتب – نسخة ويب (Frontend فقط)

✅ يعمل بدون سيرفر – مناسب للرفع على GitHub Pages.

## الملفات
- `index.html` صفحة الدخول (Admin/Manager + Employee عرض فقط)
- `admin.html` لوحة الإدارة (Admin)
- `manager.html` لوحة المدير (Manager)
- `employee.html` بوابة الموظف (عرض فقط)
- `README.md` هذا الملف

> النسخة **Self‑Contained**: كل CSS/JS داخل الصفحات (بدون assets).

## بيانات الدخول
- Admin: `admin / admin123`
- Manager: يتم إنشاؤه من لوحة Admin (قسم المدراء)
- Employee: يدخل باسم الموظف فقط (كما هو مسجّل)

## ملاحظات
- التخزين في LocalStorage داخل المتصفح.
- للنسخ الاحتياطي: من Admin > Excel (CSV) > **نسخة احتياطية (JSON)**.
- لاستعادة النسخة: ارفع ملف JSON في نفس القسم.

## Excel / CSV
تم دعم CSV مباشرة (Excel يفتحها بسهولة):
- قالب الموظفين: `الاسم, القسم, الراتب الشهري, ملاحظة`
- قالب الدوام: `التاريخ, اسم الموظف, الدخول, الخروج, عذر, نوع العذر, مشوار خارج المصنع, ملاحظة`
- تقرير الرواتب: يتم تصديره من تبويب الرواتب.

## PDF
زر "صك قبض" يفتح نافذة طباعة يمكن حفظها كـ PDF من المتصفح.
